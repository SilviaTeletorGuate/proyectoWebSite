# proyectoWebSite
Este es un proyecto web básico creado como práctica. Contiene HTML y CSS

## Cómo ejecutar el proyecto

1. Clona este repositorio:
   git clone https://github.com/SilviaTeletorGuate/proyectoWebSite.git

2. Entra a la carpeta del proyecto:
   cd proyectoWebSite

3. Abre la página principal en tu navegador:
   - Haz doble clic en `myWebSite.html`  
   o  
   - Abre con tu editor favorito y usa “Live Server” si tienes VS Code
